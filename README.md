# RickMortyMVVM
Bu uygulamada api üzerinden veri çektim.Hilt,Retrofit,Paging,Staggered grid layout kullandım.Paging yapısı ve Staggered grid yapısını bu projede öğrendim.Paging sayesinde
kullanıcı aşağı kaydırdıkça veri yüklenmesini sağlıyor.Yani sayfa sonuna gelince apiye tekrar istek atılıyor.Böylelikle bir anda uygulamadan bütün veriler gelmiyor.
Staggered grid ise gelen verinin görsel büyüklüğüne göre boyutu ayarlanıyor.Uygulamaya farklı bir tasarım katıyor.
